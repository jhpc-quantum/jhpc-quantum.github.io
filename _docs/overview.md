# Overview

JHPC-Quantumは、HPC (High-Performance Computing) と量子コンピュータを統合し、
社会実装レベルの Hybrid Computing を実現するプロジェクトです。

## Mission
- HPC/QPU 統合ソフトウェア基盤の開発
- 量子化学・最適化・AI など実アプリケーションの実証
- RIKEN R-CCS / 企業 / 大学による共同開発
- Fugaku, ABCI, GPU クラスタ、量子計算機（叡-Rei など）を統合

## コンポーネント
- HPC Backend（MPI, GPU, Fugaku）
- Quantum Backend（Gate-model, Annealing）
- Hybrid Orchestrator
- Python/C++ SDK
